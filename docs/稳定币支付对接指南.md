# Telegram Mini App 稳定币 (USDT) 支付对接指南

## 概述

在 Telegram Mini App 中接收稳定币支付，最自然的方式是走 **TON 链上的 USDT**。Tether 官方已在 TON 网络发行了 USDT（作为 Jetton 代币），它与 Telegram 生态深度绑定，用户通过 Tonkeeper、Telegram 内置钱包等即可支付。

## 支付方案对比

| 方案 | 适用场景 | 手续费 | 集成难度 | 用户体验 |
|------|---------|--------|---------|---------|
| **TON Pay SDK** (推荐) | 通用稳定币收款 | < $0.01/笔 | ★★☆ | 最佳，一键支付 |
| **直接 Jetton Transfer** | 自定义流程 | < $0.01/笔 | ★★★ | 好，需钱包确认 |
| **Cryptomus/NOWPayments** | 多链 USDT/USDC | 0.5-1%/笔 | ★☆☆ | 中等，跳转外部页 |

## 方案 A：TON Pay SDK（推荐）

TON Pay 是 TON 官方的支付层，提供完整的商户方案，支持 USDT Jetton 收款。

### 安装

```bash
# 前端
npm install @ton-pay/ui-react @ton-pay/api @tonconnect/ui-react

# 后端
npm install @ton-pay/api
```

### 前端集成

```tsx
import { TonPayButton, useTonPay } from '@ton-pay/ui-react';
import { createTonPayTransfer } from '@ton-pay/api';

function USDTPayment({ amount, orderId }) {
  const { pay } = useTonPay();
  const [isLoading, setIsLoading] = useState(false);

  const handlePay = async () => {
    setIsLoading(true);
    try {
      const { txResult, reference } = await pay(async (senderAddr) => {
        // 从后端获取支付参数（推荐）或直接在前端创建
        const { message, reference, bodyBase64Hash } = await createTonPayTransfer(
          {
            amount: amount,
            // 关键：asset 填 USDT Master 地址，而非 "TON"
            asset: 'EQCxE6mUtQJKFnGfaROTKOt1lZbDiiX1kCixRv7Nw2Id_sDs',
            recipientAddr: 'YOUR_MERCHANT_WALLET',
            senderAddr,
            commentToSender: `Order #${orderId}`,
          },
          { chain: 'mainnet' }
        );
        return { message, reference, bodyBase64Hash };
      });
      // 支付成功
      console.log('Paid!', txResult);
    } catch (e) {
      console.error('Failed:', e);
    }
    setIsLoading(false);
  };

  return <TonPayButton handlePay={handlePay} isLoading={isLoading} />;
}
```

## 方案 B：直接 Jetton Transfer（无需 TON Pay）

如果不想依赖 TON Pay 服务，可以直接通过 TonConnect 让用户发送 USDT Jetton 转账。

### 核心原理

TON 上的 USDT 是一种 **Jetton**（TON 的代币标准）。每个用户对 USDT 有一个独立的 **Jetton Wallet 合约**。转账流程是：

```
用户 → 自己的 USDT Jetton Wallet → internal_transfer → 商户的 USDT Jetton Wallet
```

### 转账消息结构

```
┌─────────────────────────────────────────────────────────────┐
│  Transfer Message (发给用户自己的 Jetton Wallet)              │
├─────────────────────────────────────────────────────────────┤
│  op: 0x0f8a7ea5 (transfer)                                  │
│  query_id: 0                                                │
│  amount: 转账金额 (USDT 6位小数, 1 USDT = 1000000)          │
│  destination: 商户 TON 钱包地址                              │
│  response_destination: 用户自己的地址 (接收余额通知)          │
│  forward_ton_amount: 0.01 TON (通知费)                      │
│  forward_payload: 订单ID (用于后端匹配)                      │
└─────────────────────────────────────────────────────────────┘
```

### 前端代码

```typescript
import { useTonConnectUI } from '@tonconnect/ui-react';
import { beginCell, Address, toNano } from '@ton/core';

async function sendUSDT(amount: number, orderId: string) {
  const [tonConnectUI] = useTonConnectUI();

  // 1. 获取用户的 USDT Jetton Wallet 地址（从后端获取）
  const { jettonWalletAddress } = await fetch('/api/payments/usdt/prepare', {
    method: 'POST',
    body: JSON.stringify({ senderAddress: wallet.account.address, amount, orderId }),
  }).then(r => r.json());

  // 2. 构造 transfer 消息体
  const body = beginCell()
    .storeUint(0x0f8a7ea5, 32)                              // op: transfer
    .storeUint(0, 64)                                        // query_id
    .storeCoins(BigInt(Math.floor(amount * 1_000_000)))      // amount (6 decimals)
    .storeAddress(Address.parse('MERCHANT_TON_ADDRESS'))      // destination
    .storeAddress(Address.parse(wallet.account.address))      // response_destination
    .storeBit(false)                                          // no custom_payload
    .storeCoins(toNano('0.01'))                              // forward_ton_amount
    .storeBit(true)                                           // forward_payload exists
    .storeRef(
      beginCell()
        .storeUint(0, 32)                                    // text comment prefix
        .storeStringTail(orderId)                            // 订单ID作为comment
        .endCell()
    )
    .endCell();

  // 3. 发送交易
  await tonConnectUI.sendTransaction({
    validUntil: Math.floor(Date.now() / 1000) + 300,
    messages: [{
      address: jettonWalletAddress,        // 用户的 USDT Jetton Wallet
      amount: toNano('0.1').toString(),     // gas 费（TON）
      payload: body.toBoc().toString('base64'),
    }],
  });
}
```

### 后端验证到账

```typescript
// 轮询 TON API 检查商户 Jetton Wallet 的入账
async function checkPayment(orderId: string): Promise<boolean> {
  const response = await fetch(
    `https://tonapi.io/v2/accounts/${MERCHANT_JETTON_WALLET}/events?limit=20`
  );
  const { events } = await response.json();

  for (const event of events) {
    // 检查 Jetton Transfer 事件中的 comment 是否匹配订单 ID
    for (const action of event.actions) {
      if (
        action.type === 'JettonTransfer' &&
        action.JettonTransfer?.comment === orderId
      ) {
        return true;
      }
    }
  }
  return false;
}
```

## 方案 C：第三方加密支付网关

如果你需要支持多条链的 USDT（如 ERC-20、TRC-20、BEP-20 等），可以使用第三方支付网关。

### Cryptomus

```typescript
// 创建支付订单
const response = await fetch('https://api.cryptomus.com/v1/payment', {
  method: 'POST',
  headers: {
    'merchant': process.env.CRYPTOMUS_MERCHANT_ID,
    'sign': generateSign(payload), // HMAC 签名
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    amount: '9.99',
    currency: 'USDT',
    network: 'tron',    // 或 'eth', 'bsc', 'ton'

    url_return: 'https://t.me/your_bot/your_app',

const { url } = await response.json(); // 用户跳转到这个 URL 支付
```

## USDT on TON 关键信息

| 项目 | 值 |
|------|---|
| 代币名称 | Tether USD (USDT) |
| 标准 | Jetton (TEP-74) |
| 精度 | 6 位小数 (1 USDT = 1,000,000 最小单位) |
| Mainnet Master 合约 | `EQCxE6mUtQJKFnGfaROTKOt1lZbDiiX1kCixRv7Nw2Id_sDs` |
| Testnet Master 合约 | `kQD0GKBM8ZbryVk2aESmzfU6b9b_8era_IkvBSELujFZPsyy` |
| 转账 gas 费 | ~0.05-0.1 TON (< $0.30) |
| 确认时间 | 0.6-6 秒 |
| 查询 API | tonapi.io / toncenter.com |

## 你需要准备的

1. **TON 钱包地址** — 用 Tonkeeper 或 MyTonWallet 创建一个（用于收款）
2. **TON Pay 商户账号**（方案 A）— 在 pay.ton.org 注册
3. **少量 TON** — 用于激活钱包和测试（testnet 可从水龙头免费获取）
4. **Testnet USDT** — 从 testnet.tonscan.org 的水龙头获取

## 测试步骤

```bash

```
